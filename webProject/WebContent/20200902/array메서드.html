요소 찾기

arr : "사과","바나나","참외","복숭아","자두","수박","사과","바나나"];
index of(item, start) : item의 위치를 찾는다
lastIndexOf(item, start) : 마지막 item의 위치를 찾는다

배열에 추가/삭제

arr : "사과","바나나","참외","복숭아","자두","수박","사과","바나나"];
push(item) : 배열의 맨 마지막으로 추가
pop() : 배열의 마지막 요소를 제거

배열에 추출과 수정

arr : "사과","바나나","참외","복숭아","자두","수박","사과","바나나"];
slice(start,end) : start ~ (end -1)까지의 요소를 추출한다
splice(idx,n,a,b,c): idx의 위치에서 n개를 삭제하고 a,b,c를 추가한다

배열합치기와 문자열 변환

arr : "사과","바나나","참외","복숭아","자두","수박","사과","바나나"];
arr. concat(a) : arr배열과 a배열이 합쳐져서 하나의 배열로 표현한다
arr. join() : arr배열을 문자열로 변환한다
